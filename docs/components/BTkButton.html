BTkButton

Interactive Button Component

The BTkButton component provides modern, stylish buttons with various styles, hover effects, and customization options.

Modern Design: BTkButton is a modern replacement for tkinter's Button widget, featuring built-in styling, smooth animations, icon support, and fully customizable appearance.

üöÄ Quick Start

Python 
from bettertkinter import BTk, BTkButton

app = BTk()

# Simple button
button = BTkButton(app, text="Click Me!", command=lambda: print("Clicked!"))
button.pack(pady=10)

app.mainloop()

üí° Examples

Complete Button Example

Python 
import tkinter as tk
from bettertkinter import BTk, BTkButton, BTkFrame

class ButtonDemo:
    def __init__(self):
        self.app = BTk()
        self.app.title("BTkButton Demo")
        self.app.geometry("600x400")
        
        self.setup_ui()
        
    def setup_ui(self):
        # Main frame
        main_frame = BTkFrame(self.app, corner_radius=20, fg_color="#2B2B2B")
        main_frame.pack(pady=20, padx=20, fill="both", expand=True)
        
        # Title
        title = tk.Label(
            main_frame, 
            text="Button Styles Demo", 
            font=("Arial", 20, "bold"),
            bg="#2B2B2B", 
            fg="white"
        )
        title.pack(pady=20)
        
        # Button container
        button_frame = BTkFrame(main_frame, fg_color="transparent")
        button_frame.pack(pady=20)
        
        # Different button styles
        self.create_buttons(button_frame)
        
    def create_buttons(self, parent):
        styles = [
            ("Default", "default"),
            ("Primary", "primary"), 
            ("Success", "success"),
            ("Danger", "danger")
        ]
        
        for i, (text, style) in enumerate(styles):
            btn = BTkButton(
                parent,
                text=text,
                style=style,
                width=120,
                height=40,
                command=lambda s=style: self.button_clicked(s)
            )
            btn.grid(row=i//2, column=i%2, padx=10, pady=10)
            
    def button_clicked(self, style):
        print(f"{style.capitalize()} button clicked!")
        
    def run(self):
        self.app.mainloop()

if __name__ == "__main__":
    demo = ButtonDemo()
    demo.run()

‚ú® Best Practices

Design Guidelines

  • Use consistent button sizes within your application
  • Choose appropriate styles based on button importance (primary, secondary, etc.)
  • Provide clear, action-oriented text labels
  • Consider using icons for frequently used actions
  • Ensure sufficient contrast between text and background colors

Performance Tips

  • Avoid creating too many buttons with complex hover animations
  • Use image icons sparingly to maintain smooth performance
  • Consider button pooling for dynamically created interfaces
